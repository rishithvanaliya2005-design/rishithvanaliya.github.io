Automated CFD Simulation Platform

Full-Stack OpenFOAM Automation with Python & Flask

Project Overview

Developed a lightweight full-stack platform that automates OpenFOAM CFD workflows on Linux. Integrated Python–Bash pipelines for running blockMesh, checkMesh, and icoFoam, enabling streamlined simulation setup, execution, and real-time log visualization through a web interface.

What It Does

The platform allows users to upload an OpenFOAM case, automatically triggers meshing and solver execution, and returns the simulation output and post-processed results. It eliminates the need for manual command-line steps and ensures consistent, repeatable runs through an isolated execution environment.

Key Features

  • Automated OpenFOAM workflow (blockMesh → checkMesh → icoFoam).
  • Python–Bash execution pipeline with structured logging.
  • Web interface built with Flask for case upload and monitoring.
  • Containerized environment ensuring identical results across machines.
  • Validated on a cloud VM for remote simulation execution.
